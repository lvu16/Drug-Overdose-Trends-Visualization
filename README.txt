DRUG OVERDOSE TRENDS VISUALIZATION
This interactive data visualization project presents U.S. drug overdose death rates by sex, race, age, and drug type from 1999 onward. Built using D3.js, the goal is to make CDC data more accessible and actionable for public health officials, policy makers, nonprofits, and educators.

Data: Drug overdose death rates, by drug type, sex, age, race, and Hispanic origin: United States
Publisher: Centers for Disease Control and Prevention
Maintainer: National Center for Health Statistics
First published: 04/28/2022
https://catalog.data.gov/dataset/drug-overdose-death-rates-by-drug-type-sex-age-race-and-hispanic-origin-united-states-3f72f


🚀 Features
Bar Chart: View overdose trends across gender and race

Line Chart: Analyze trends by age groups over time

Grouped Bar Chart: Compare death rates by drug type

Interactive Dropdowns: Select categories and chart types dynamically

Tooltips & Legends: On-hover details and color-coded keys

Clean UI: Responsive and accessible design using HTML/CSS

📁 Project Structure
.
├── index.html          # Main HTML file with dropdown and chart container
├── main.js             # JavaScript logic using D3 to render charts
├── style.css           # Styles for layout and visuals
├── drugoverdose.csv    # Data file from CDC
└── README.md           # Project instructions and documentation

💡 How to Use
Clone or download this repository.

Open the folder in VSCode or any code editor.

Double-click index.html to launch it in your default browser.

✅ No server setup or external dependencies required. Everything runs locally in the browser.
